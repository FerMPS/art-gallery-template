# Digital Gallery Template

> **Estética Renacentista en la Web Moderna.**

Este proyecto es un **Template de Exhibición** diseñado bajo una curaduría visual de alto nivel. Está pensado para artistas y marcas de lujo que buscan una narrativa digital donde el espacio y la obra sean los protagonistas.

---

## Filosofía de Diseño

El sistema visual se basa en tres pilares fundamentales:

* **Elegancia Minimalista:** Uso estratégico de espacios en blanco y una paleta de colores sobria (`#383a3d`, `bg-gallery-bg`) para resaltar la cromática de las obras.
* **Tipografía Curada:** Contrastes entre fuentes *Thin* y *ExtraLight* para evocar la delicadeza del trazo clásico, manteniendo una legibilidad impecable.
* **Interactividad Inmersiva:** * **Micro-interacciones:** Efectos de *hover* con escalado suave y desenfoques (*backdrop-blur*) que aportan profundidad.

---

## ADN Técnico

* **Framework:** Laravel 11 + Blade Components.
* **Estilos:** Tailwind CSS v4 (Utility-first para diseño de precisión).
* **Animaciones:** AOS.js (Scroll revelations) y transiciones nativas de alta duración.
* **Arquitectura:** Lógica centralizada en un Layout único para una navegación fluida.

---

## Propuesta de Valor

Este es un **template personalizable**. La estructura está lista para ser adaptada a cualquier colección privada o portafolio profesional, manteniendo la jerarquía visual y la sofisticación técnica original.
